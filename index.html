About

What’s in this repository

Every exercise is built to the same conventions

  • One self-contained HTML file — no external scripts, fonts, or CDNs, so it works offline and inside a Canvas iframe.
  • CSP-safe: all behavior wired through event listeners, nothing inline.
  • WCAG 2.1 AA — semantic landmarks, a skip link, keyboard-reachable controls, and visible focus.
  • Light and dark themes that follow the reader’s system setting.
  • A collapsed instructor answer key at the end, clearly fenced so it can be deleted before distribution.

The exercises reference the Simulated SOC — a four-screen SIEM, network, incident-response, and threat-intel environment.
